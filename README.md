# ------------------------------------------------------
# Overview
# Chromatic Drift is a minimalist photography website that curates nature and street photo collections, 
# invites collaboration from other photographers, and makes it easy for people to get in touch for 
# projects, licensing, or partnerships.

# ------------------------------------------------------
# Features
# - Curated Galleries: Dedicated pages for Nature and Street photography, with title, story/caption, 
#   and location/device details.
# - Collaboration Program: Contributor guidelines with attribution-first principles.
# - Contact Page: Human-friendly contact details for collaborations, licensing, workshops.
# - Mobile-Friendly Layout: Clean typography, accessible navigation.
# - Static & Fast: Deployed as a static site for speed and reliability.

# ------------------------------------------------------
# Tech Stack
# - Frontend: Static site (HTML, CSS, minimal JavaScript)
# - Hosting: Netlify
